# SpendWise | Premium Expense Tracker

SpendWise is a modern, responsive personal finance management application designed with a premium **Glassmorphism** aesthetic. It provides users with deep insights into their spending habits, interactive budget tracking, and full customization across multiple languages and currencies.

![SpendWise Showcase](https://raw.githubusercontent.com/tustelculture-web/Mobileexpensetrackerapp/main/public/showcase.png)

## ✨ Features

### 🚀 Smart Budgeting
- **Interactive Allocation**: Set and update your monthly budget directly from the dashboard.
- **Real-time Tracking**: Monitor your remaining balance as it updates with every transaction.
- **Monthly Reset**: Automatically calculates spending for the current month and tracks days remaining.

### 📊 Advanced Analytics
- **AI-Powered Insights**: Get automated alerts for high spending, great progress, and budget status.
- **Category Comparison**: Compare current month spending against the previous month with dual progress bars.
- **Top Categories**: Automatically identifies your "Best Category" (most saved) and "Needs Attention" (highest overspend).
- **Trend Charts**: Beautiful, interactive spending trend visualizations.

### 🎨 Premium UI/UX
- **Glassmorphism Design**: Elegant translucent layers for a modern, state-of-the-art feel.
- **Dynamic Themes**: Seamlessly toggle between **Light** and **Dark** glassmorphism modes.
- **Fully Responsive**: Optimized for both mobile devices (bottom navigation) and desktop (sidebar).

### 🌍 Localization & Customization
- **Multi-Language**: Full support for **English** and **Bahasa Indonesia**.
- **Multi-Currency**: Toggle between **USD ($)** and **IDR (Rp)** with locale-specific formatting.
- **Category CRUD**: Full ability to Add, Edit, and Delete categories with custom icons and brand colors.

### 💾 Data Management
- **Export Data**: Download all transactions to **CSV** for external analysis.
- **Cloud Sync**: Simulated integration with **Google Sheets** for seamless data backup.
- **Local Persistence**: All data and settings are saved locally in your browser.

## 🛠️ Tech Stack

- **Framework**: [Vite](https://vitejs.dev/) + [React](https://reactjs.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Charts**: [Recharts](https://recharts.org/)
- **Components**: [Shadcn UI](https://ui.shadcn.com/)
- **Date Handling**: [date-fns](https://date-fns.org/)
- **Notifications**: [Sonner](https://sonner.emilkowal.ski/)

## 🚀 Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/tustelculture-web/Mobileexpensetrackerapp.git
   cd Mobileexpensetrackerapp
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

## 🌐 Deployment

The app is ready to be hosted on [Vercel](https://vercel.com/):

1. Push your code to GitHub.
2. Import the repository into Vercel.
3. Vercel will automatically detect the Vite settings and deploy.
